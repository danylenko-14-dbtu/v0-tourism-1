# Структура Проєкту

## Коротко
Проєкт побудований на `Next.js App Router` і складається з 5 основних частин: сторінки, секції інтерфейсу, словники локалізації, службова логіка та статичні ресурси.

## Основні Папки
- `app/` — маршрути, сторінки, layout-и та глобальні стилі.
- `components/` — React-компоненти сайту.
- `components/sections/` — основні секції лендінгу: hero, professions, program facts, apply, pricing, partners, FAQ, footer.
- `components/ui/` — базові UI-компоненти та примітиви.
- `dictionaries/` — текстовий контент для локалей `uk` і `en`.
- `lib/` — допоміжна логіка: i18n, завантаження словників, utils, public env.
- `public/` — статичні файли: логотипи, фото команди, відео hero, favicon, PDF-документи.
- `styles/` — додаткові стилі та CSS-модулі.
- `hooks/` — окремі хуки, якщо потрібна винесена клієнтська логіка.

## Ключові Файли
- `app/[locale]/page.tsx` — головна сторінка локалізованого сайту.
- `app/[locale]/faq/page.tsx` — окрема FAQ-сторінка.
- `app/[locale]/layout.tsx` — layout для локалізованих маршрутів.
- `lib/i18n.ts` — конфігурація локалей.
- `lib/dictionaries.ts` — підключення словників.
- `dictionaries/uk.json` — основний український контент сайту.
- `dictionaries/en.json` — англомовна версія контенту.
- `SITE_CONTEXT.md` — короткий опис і сенс сайту для LLM.

## Як Читати Проєкт
- Якщо треба змінити структуру сторінок: дивитися `app/`.
- Якщо треба змінити контент секцій: дивитися `dictionaries/*.json`.
- Якщо треба змінити вигляд або поведінку блоку: дивитися `components/sections/`.
- Якщо треба змінити спільні елементи інтерфейсу: дивитися `components/ui/`.
- Якщо треба оновити медіа або документи: дивитися `public/`.
